RMSOFT MiniSHop
Module to create online catalogs
http://www.redmexico.com.mx
==========================================

REQUIREMENTS
------------------------
  - RMSOFT Common Utilities 1.3
  - GD2


INSTALATION
------------------------
 - Upload the "rmms" folder to your XOOPS INstalation "modules" directory.
 - Go to XOOPS Control Panel and install it.
